# C++ Projects

Dieses Repository enthält mehrere kleinere Projekte aus meinem Informatikstudium.

## Inhalte

### Oasencrawler
2D Konsolen-Spiel mit rundenbasierter Logik und einfacher Gegnerbewegung.  
Fokus auf grundlegende Spielmechaniken und objektorientierte Programmierung.

### String-Klasse
Eigene Implementierung einer String-Klasse in C++.  
Ziel war es, das Verständnis für Speicherverwaltung und Klassenstruktur zu vertiefen.

## Technologien
- C++
- Objektorientierte Programmierung (OOP)

## Hinweis
Die Projekte wurden im Rahmen des Studiums erstellt und dienen zur praktischen Anwendung von Programmiergrundlagen.
